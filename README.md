# Predictive-ML-Pipeline
Engineered a dual-stage pipeline following the CRISP-DM framework to automate loan approval classification and
estimate maximum lending values.
– Developed and optimized Logistic Regression, Na¨ıve Bayes, and KNN models, integrating them into a probability-based
Voting Ensemble.
– Utilized GridSearchCV for hyperparameter tuning and compared fully grown vs. level-pruned Decision Tree Regressors to
optimize accuracy.
– Evaluated performance using confusion matrices, AUC-ROC curves, and regression metrics (R2, MSE, MAE) against
banking success criteria.
